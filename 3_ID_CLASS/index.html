ID E CLASS

ID

O ID funciona como um CPF. Assim como só posso ter um número de CPF,cada elemento só pode ter um valor de ID. O CPF é único, então não é possível repetir o memso valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que você só pode ter um valor, no CLASS você pode repetir um valor, no CLASS você pode utilizar o mesmo valor para vários elementos.
